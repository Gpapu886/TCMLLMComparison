# Data availability note

This repository provides code, task-file lists, evaluation scripts, leakage-audit summary statistics, processed non-identifying result tables, bootstrap confidence intervals, and MedBench task-composition results supporting the manuscript.

The complete raw clinical medical-case text is not redistributed here. The medical-case data were derived from a third-party public resource, and the raw text may include clinical case descriptions. To avoid inappropriate redistribution of third-party clinical text, only processed non-identifying summaries and result tables are included.

Researchers who wish to reproduce the full pipeline should obtain the original JSON resources from:

https://ai.gitee.com/ljt365fir/SmartSage-ZYLLM-32B

After obtaining the source data, the evaluation scripts in this repository can be used to reproduce the reported metrics.
